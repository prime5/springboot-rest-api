# Spring Boot REST API

A simple RESTful web service built with Spring Boot. This project demonstrates how to create a basic API with endpoints for handling HTTP requests.

## Features

- Spring Boot 3.x
- RESTful endpoints
- Maven build system
- Java 17+
- Easy to extend

## Getting Started

### Prerequisites
- Java 17 or higher
- Maven 3.x
